Conception et implémentation d'une base de données

Modélisation et création de la base de données de GreenSD, entreprise de logistique bas-carbone.

Retour aux projets
👥 Travail en équipe BDD SQL Modélisation
Description du projet

Cette SAE avait pour objectif d'informatiser le suivi des livraisons, des tournées, des conducteurs-livreurs et de l'empreinte carbone de chaque colis pour GreenSD, une entreprise fictive spécialisée dans le transport et la logistique bas-carbone.

Compétences mobilisées
  • Modéliser une base de données (MCD/MLD)
  • Normaliser un schéma relationnel
  • Implémenter une base de données sous un SGBD
  • Écrire des requêtes SQL de création
  • de peuplement et d'interrogation
  • Concevoir une application fonctionnelle s'appuyant sur la base
Mon travail

Dans le cadre de ce projet, j'ai participé à la modélisation de la base de données de GreenSD, en construisant le modèle conceptuel (MCD) puis le modèle logique (MLD). Nous avons ensuite implémenté ce schéma sous un SGBD relationnel, peuplé les tables et écrit des requêtes SQL permettant de répondre aux besoins de suivi de l'entreprise.

Portfolio réalisé dans le cadre du BUT Science des Données · IUT de Poitiers, campus de Niort · 2024–2025
